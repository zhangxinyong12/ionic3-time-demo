### ionic3-demo
 [具体请多多看看ionic官网](https://ionicframework.com/docs)
### 项目初始化
 - 安装node,下载稳定版 8.11.4， [官网下载](https://nodejs.org/zh-cn/)
 
 - 全局安装 ionic 

     `
       npm install -g ionic@3.9.2
    `
- 下载依赖

   `
    npm install
   `
- 启动项目
   
   `
    ionic serve
   `
- 打包网页版

    `
        ionic build --prot
    `
    
    www目录下面的即是。
